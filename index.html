Welcome to the page! This is a small project which is built using MongoDB and Node JS which creates a Hospital Management System. The front-end is constructed using HTML, CSS and Javascript and is connected to the MongoDB database using Node JS. We can perform the basic CRUD (Create, Read, Update and Delete) operations on our database. Click around everywhere to get a thourough understanding of the page!

Note: The "Show Database" button also works as the refresher for the database so please click on it every time you modify the database to reflect all of the changes.
